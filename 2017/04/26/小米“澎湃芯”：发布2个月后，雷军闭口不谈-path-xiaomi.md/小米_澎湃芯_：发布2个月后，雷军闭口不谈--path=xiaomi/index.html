小米“澎湃芯”：发布2个月后，雷军闭口不谈--path=xiaomi

7年而立的小米，还能玩得转互联网这个大模式吗？

小米6依然在万众瞩目中拉开了它的神秘面纱。

“小米6是小米历时7年，在工艺上的巅峰之作”。小米公司董事长兼CEO雷军掷地有声地说到。而在7年前，小米才刚刚成立。

在小米1发布时，随着雷军那句经典的“没有设计就是最好的设计”被“流传千古”，小米公司逐渐被人皆知。从没有设计就是最好的设计到如今的小米6被雷军誉为“巅峰之作”，小米能否扭转颓势在登高峰？

雷军在小米6的发布会上表示，除了颜值外，在性能上也有所突破，其采用的是高通公司的最新处理芯片——骁龙835。但是，产能依旧是摆在小米面前的一座大山。

有业内人士透露，当前小米6仅有8万库存，而光发布当天的预约人数就已超过80万，是库存数量的10倍之多。如果能够满足80万人，至少还需要生产8个月左右，或许到小米7发布之前，货源供应才能稳定下来。

另外，在此次发布会中，雷军却没有谈及已发布2个多月的自家处理器芯片“澎湃”，这不得让人琢磨，难道自家芯片遇到了某种问题？

同时，雷军还在发布会提及，小米已改变为线上线下相融合的新零售模式，但是转型又能给小米带来什么机会呢？

如今7年而立的小米，还能玩得转互联网这个大模式吗？

绝口不提“澎湃”芯片

今年2月28日，小米正式发布小米松果处理器，这让雷军激动万分。当时，雷军称，从做芯片之时起，到现在已经烧了10亿元。对于回本，小米芯片一年至少要卖几千万时才能打平。

雷军说，小米之所以做芯片，在于小米的出货量优势。“如果一家公司从0开始单独做芯片，要说服手机公司用芯片是很难的事，因为不知道你是怎么做的，用你的芯片是一件很冒险的事。”

目前，从全球智能手机产业来看，只有3家拥有自己的芯片，分别为苹果、华为和三星。

苹果做处理器用了8年时间，三星做处理器同样也用了几乎相同的时间，而华为做处理器却用了近10年。

对于用了10年研发芯片的华为，为何小米只用了两年就做出芯片？雷军说，“华为做芯片是十几年前的事，那个时间点做芯片需要的周期注定很长。但现在基础技术更成熟了，我们做起来就很快，这叫后发优势。”

然而，在后发优势的2个月后，雷军却只字未提“澎湃S1芯片”的产能和销量究竟如何，却是一味的在吹嘘小米6的功能有多么的先进，骁龙835的处理性能有多么的快速。

对于自己的“处理器孩子”，他选择了沉默。当然，选择沉默或许是因为他还没有准备好该如何开口。

小米5C在市场上的销量并不怎么好。“小米5C有货，但需要加100元。”中关村一位手机摊主表示。

“小米5C的机器基本没有多少消费者买，即便买也是尝鲜的人居多，毕竟是第一代小米芯片，还是有些不稳定，发热卡顿导致的退货也很多。”

还有店主表示，已经积压了很多小米5C的货源，打算原价处理。而对于众人吐槽的小米5C只能用移动网，该店主表示，可以破解联通网络。“现在只能破解成移动联通双4G，不然（光）单网，买的人更少。”

因为，目前小米松果的澎湃S1处理器仍徘徊于较低端的A53架构，在制作工艺上，其采用台积电28纳米工艺，与高通骁龙625处理器采用的三星14纳米工艺、华为麒麟655采用的台积电16纳米工艺相差一代，和联发科水平相当。而华为的海思麒麟处理器已从920升级到960，其架构也用上了主流的A73。

在基带方面，小米澎湃S1最高只支持到五模LTE Cat.4，属于较低阶级别。当前高通815和华为海思麒麟960均已达到九模LTE Cat.6，这也是为何小米5C却只能用移动联通卡的原因。

而本次小米6所搭载的骁龙835应该是高通近几年出品的一颗综合性能最好的处理器芯片。从跑分来看，整体性能相对骁龙821有大幅提升。业界认为，能够与其相比较的唯有华为麒麟960处理芯片。

所以，雷军很聪明，他不会把“澎湃”芯片搭载到小米高端机之上，因为一旦出现问题，就会造成不可挽回的声誉。我们可以看出的是，小米5C依然扮演者“试验者”的身份存在于市场之中。

目前，在京东商城的小米专营店内，依然没有小米5C的身影，反而在第三方卖家的店铺里货源很充足，销量基本在3000台左右。

相比第三方卖家能够第一时间拿到现货，小米官网却依然需要抢购。曾在官网进行抢购，却发现不到3秒钟就已没货，可见产能严重不足。

再提“新零售”

那么，小米6究竟能挽回小米的颓势吗？或许未必。

首先，在外观设计上，小米6的设计与华为荣耀Magic和荣耀8趋同，四面曲屏也并非小米首创。四周曲面屏的做法早在荣耀Magic身上有所尝试，但因为工艺问题，荣耀Magic至今产能刚刚有所提升。

其次，在摄像头设计上，小米6采用了横排双摄技术，而其算法采用的和苹果一致，为广角+长焦。除了苹果，该算法还在金立M2017上被采用。但该方案并非对画质产生提升效果，只是在对焦远处时会稍微清楚些，远不如彩色+黑白的相机算法。

另外，小米6取消了耳机孔的设计，取而代之为利用数据线接口，类似苹果的处理方法。但此举已经在苹果时代被众多网友所吐槽。

本次小米6定位在了2499元的价格，可以说算得上是中端机型，销量应该不是什么问题，但唯一的缺点则是小米的新品通常会被“耍猴”，也就是我们常说的缺货严重。

2015年，小米没能如期完成8000万的出货目标，2016年小米更是无缘销量前5名。

有业内分析人士认为，小米销量的下滑或许大多数原因在于其经常缺货。毕竟华为、oppo、vivo的新品并不用等待很长时间，基本都是现货供应。因此，这是摆在小米面前的一道坎。

如今的手机市场发生了翻天覆地的变化，它不在是7年前的那个风口，如果在回到7年前，雷军的高配低价战略或许还有所奏效。

随着2015年开始，华为、oppo、vivo的强势崛起。小米只有互联网环境，也只有线上市场，但线上市场的潜力终被小米做完了。随后两年，中国智能手机市场变成华为、oppo和vivo的天下，因为他们反攻线下，延伸线上，而小米的线下之路却进展缓慢，线上却步步后退。

因此，雷军像马云一样，提出了“新零售”概念。雷军表示，小米已从之前的纯电商、线上销售，改变为线上线下相融合的新零售模式，要开拓小米“新零售”版图。而1000家小米之家，是雷军所定的“小目标”。

在此前的小米发布会中，雷军曾提及“线下开店”。 雷军称，小米遇到的真正挑战是，当前电商模式只占整个市场的20%左右，线下渠道依然很强势。

应对线下渠道带来的挑战，雷军思考了两种方式，第一种方式是守住电商就行了，第二种方式是用互联网思维打造高效的零售模式，小米采取了后一种。

如今，智能手机行业发展到现在，所谓的互联网思维也有了饥饿营销的贬义色彩，因为oppo和vivo两家拥有大量线下传统店的崛起，让小米这家互联网手机公司，再次重新审视“线下渠道”。

随着互联网手机“下半场”的到来，当互联网企业急于开拓模式创新时，轻视线下渠道让他们在无形之中吃了亏，这一点在手机行业尤其明显。 因此。小米对此采取的措施是——加强供应链和增加线下渠道。

有业内人士分析称，“互联网思维是雷军的线上饥饿营销，不注重线下产，被蓝绿大厂赶超，认为雷军走了互联网的极端。”很显然，互联网的上半场是线上为王的时代，但互联网的下半场却是供应链和渠道为王的时代。

这也从中佐证了，无论任何互联网线上模式，最终都需要走到线下，走向实体。

现在，小米全面落后华为等厂商，雷军和小米才算是彻底醒悟，要搞“线下”，也就是雷军口中的“新零售”。

出海遇到困境

今年1月31日，雨果·巴拉在担任小米全球副总裁三年半之后，重返硅谷，他曾于2013年开始负责小米的国际扩张。

巴拉在小米任职期间，虽推动了公司设备在印度市场的销售，但该公司在巴西和全球其他市场的发展却极为有限。

市场调研公司IDC设备研究副总裁布莱恩·马表示，巴拉为打造小米的国际思维做出了大量的工作。但是看看数据，就会发现小米的国际扩展战略喜忧参半。

Counterpoint印度研究总监尼尔·沙哈认为，小米在印度市场或已接近触顶。“印度市场的网络销售存有局限性。目前，60%至70%的智能手机仍通过线下销售。中国的厂商oppo和vivo目前已在印度市场超过了小米。”

2015年，小米进军巴西市场，但随后其关闭了巴西办事处。有分析师指出，“小米一直在巴西亏本销售手机。除非在巴西组装产品，否则将面临大约60%的税款。”

这似乎不是一个让小米听起来的好消息，但这的确是存于市场的现状。而这些现状除了销量和份额，挡在小米面前的还有专利问题。

小米公司高级副总裁王翔在接受媒体采访时说，专利是以前小米快速增长时期没精力做的事，现在有能力了，正是“补课”的时候。

“虽然专利集中在大陆，缺少核心专利这种中国企业的通病依然存在，别人发明了一个东西 ，使用付费很正常，专利碰撞无法避免，只能自行适用。但小米正尝试用其他的办法解决这些问题，例如与微软合作达成重要专利交叉许可，收购专利等。”王翔说到。

而近日，王翔再次接受媒体采访，他表示，小米暂时不在美国卖手机。“虽然我们没在美国或者西欧销售，我仍然感到很抱歉，因为我们未能为他们提供很好的服务。”

对于供应链与人力资源问题，王翔表示小米目前已经在30个地区提供服务，大多数都是发展中市场。他还说：“我们追求的市场是大众市场，想造成巨大的影响，而不是关注高溢价、精英市场。我们想为每一个人创新。”

但是，这有些出尔反尔。王翔一面在说小米不关注高溢价产品，但小米又在做着高溢价产品，这不得不让人怀疑王翔所说的话。

对于不进军美国市场，王翔给出的结论是，美国智能手机市场是由运营商驱动的，如果想将设备放进它们的网络，需要定制，需要进行大量的测试。而小米的工程团队“超级忙”，很难为这些运营商提供服务。

“超级忙”便成了无法进军美国的推辞。可以见得，小米的工程技术团队人员空缺依然很大。

在小米负责全球业务的雨果走后，突围国际市场又是一道摆在小米面前的题。其实，无论是专利还是高端品牌，小米最重要的是夯实基石。唯有夯实基石才能带来更多的高端品牌和满足更多的市场需求。

所以，可以肯定的是，想打出国门还需自己的真本事。
