微信小程序

中国人最喜欢访问的网站

之前就有看到微信小程序的内测，如今微信这么受欢迎，很多开发者和微商也很关注微信的新功能，作为喜欢探测新鲜事物的用户来说，也非常期待微信小程序的进展，毕竟对喜欢网页的网友来说，更添一点色彩，总之，web程序员的春天来了。

微信是使用JavaScript网络脚本语言，被数百万计的网页用来改进设计、验证表单、检测浏览器、创建cookies，以及更多的应用，是因特网最流行的脚本语言，当然，微信还起了个名字叫welang语言，因为里面很多自定义函数组件都需要用到指定的代码，不过看那些开发者尝鲜后都说开发工具挺爽，简单易用，传送门。

如果你想体验微信小程序的话，你必须下载最新的微信，然后在朋友圈搜索小程序，就可以体验一下官方提供的Demo开发，当然，你也可以搜索其他小程序，比如搜索：豆瓣评分，还有一些你能想象得到的app市场程序。

本来想使用微信小程序来连接自己的博客，目前也没看到有人这么做，无奈下载了开发工具无法运行在winxp（又被我改回XP囧），提示错误，但我在虚拟机win7里运行居然关闭系统了，NB啊~ 目前还无法解决，只能先观望了，我是在想是否能微信用户在小程序里直接使用微信账号评论，这应该没问题吧！好吧，本人也是菜鸟，坐等有人研究代码拿出来分享哈！

当然，微信小程序的存在不仅仅是为那些微商服务，还有一个好处是可以提供一个简单方便的共享互通的平台，比如android和ios系统的互通，开发者不用独立开发一个APP，小程序的难度及成本更低了。

举个简单例子，比如我平时会用到豆瓣评分，那么通常你需要打开网页去搜索，又或者下载一个APP，但APP里面有很多你不需要的功能，也许你也不希望安装app占用你的手机空间，又或则没有广告和后台程序的干扰，你或许会说，那为什么不用网页去搜索啊，但是登录方面和操作的便携性显然没有微信里面直接操作来的方便，不得不说，微信小程序也在冲击着APP市场。

好吧，我也是昨天刚尝试了微信小程序，当然或许你会高估了它，它毕竟只是提供方便的小程序，大家有什么想法也可以交流下。
