司马迁为何没有记载韩信是如何打败项羽的？

因为司马迁写的是刘邦如何打败项羽，垓下之战只是整场战役中其中一个环节，韩信破龙且也是整场战役中一个环节。就好比淮海战役中的后期阶段杜聿明被围，只是整场战役一个环节，而韩信只是作为垓下之战汉军的临时总指挥官。

刘邦破项羽战役的过程是刘邦约韩信、彭越夹击项羽，刘邦主力先到固陵，被楚军大破。于是刘邦听信张良之计，大赏韩信、彭越、英布三王，刘贾连同叛变楚军的周殷和英布由九江北上，韩信南破龙且，留曹参在齐地，自己率军南下与刘邦会师，彭越部队亦至。

项羽原本在陈地与刘邦对持，当韩信、彭越南下，英布等北上之际，令项羽不得不后退。韩信南下作战的细节全部写在《灌婴列传》中，灌婴以韩信的先锋身份开路，连破公杲 、项声、薛公、郯公、项佗、周兰诸将，直破项羽都城彭城。项羽回不了彭城，唯有往南逃，之所以在垓下决战，大概是被北上的英布和南下的韩信堵截，被封死在垓下。战斗打到这个地步，项羽早就已经输了，只不过项羽军战斗力仍在，得选择有效方法彻底击溃这支劲旅。结合项羽本纪和高祖本纪，也就是汉军初期与项羽交战不利，韩信重整攻势进攻。项羽坚壁防守到粮食用尽，汉军在四面奏起楚歌，此时楚军大概正在逐渐溃散，项羽则亲率亲兵突围，韩信等顺理成章消灭楚军。杜聿明在淮海战役让蒋公瞎指挥被围后，不也支持作战了一个多月吗？垓下之战就是一场很典型的包围战。

经常有人黑刘邦不懂军事，但这场战役真是很成功的战略包围作战，因此得到十面埋伏的典故。打赢项羽的是整个战略布局，韩信如何在垓下击败项羽并不是很重要。
