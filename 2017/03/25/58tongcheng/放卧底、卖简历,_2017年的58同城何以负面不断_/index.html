58同城

放卧底、卖简历, 2017年的58同城何以负面不断?

近日，58同城公布了截止2016年12月31日第四季度和全年未经审计的财务报告。根据财报显示，以非美国会计师准则（non-GAAP）计算，第四季度归属于58同城公司的净亏损为70万美元，而去年同期净亏损为4850万美元。

从营收结构来看，58集团的招聘、房产、汽车、黄页及二手物品交易等传统及创新业务的收入结构更趋合理化，其中招聘版块收入加速增长。为改善亏损，未来58同城也许会将重点放在招聘版块上。

但是，没过几天，58同城就被爆出简历泄密，在淘宝上，购买数量达到10万以上，就只需要两毛钱一条，甚至于700元就能拿到一套专门采集58数据的软件。归根究底还是58给企业或者合作商的权限过大，保密措施不够完善，导致爬虫有机可趁，盗取用户信息。甚至有网友评论，这一类的企业还主动出卖用户信息给保险公司。

实际上，58同城风波不断，说到底还是因为利益，不禁想问姚劲波是想赚钱想疯了吗？

长期以来，58同城凭借着自主、免费的信息分类获得了大量的流量，尽管姚劲波一直强调自己只专注于信息展示，并且严打虚假信息。但是现实似乎打了他的脸。

有知乎网友回答称，在58有一个招聘服务，往往都是中介进行购买，并且还是打4-5折，之后这群中介就会大量地去刷屏忽悠人。甚至于还有58员工为他们伪造证件去办理会员。淘宝刷单然后被骗的人数不胜数，甚至有人质疑58同城就是依靠这群人来获得收益。

的确，58最大的两个收入来源就是会员付费和在线推广，就是姚劲波所倡导的。

同时，就租房来说，在58同城和赶集网上是没有任何审查机制的，只有当你的价格低于或者高于市场价格时会被警告。而这群人，自然也就是58最受欢迎的会员们！传统市场的眼见为实在网络上并不可行，仅仅是图片并不能保证真实性。毕竟，有了PS，谁知道你是人是鬼。而且，以租房来说，不能保证房源真实性的同时，连房主到底是干啥的都成问题。

还没来北京时，想在58同城上先找找房子，看着很精美的图片以及非常具有迷惑性的文字，都让我心动。“房主”称自己为了不被无聊的人打扰，所以只留微信不留电话，想想觉得也有道理，加了微信却发现不是那么回事。只是一个做微商的也就罢了，就怕遇见称房子没有了，然后给你介绍条件特别差的房子，或者纯粹是中介假冒房主，以各种名义收取押金。

久而久之，没有约束的58就成了骗子的集合地。

前不久，一起官司将58同城牵涉其中。其系公司查博士被竞争对手车鉴定给告了，理由是查博士派遣一名卧底到车鉴定中窃取用户资料，并且通过骚扰用户、或者报低价来扰乱市场，抢夺客户源。

在卧底的口中，我们得知了一个名字，而该人竟然是58的一个高层。后来，查博士负责人丛林以个人身份向车鉴定道歉，同时又称，自己将去58上任，只有姚劲波能代表查博士。

尽管58回应查博士的运营并不归58同城管，但是无论是58的高层，还是姚劲波的股份，都足以让大家将二者联系到一起。

连续两年的亏损，58同城的营收持续降低，加上58为了寻求新的流量入口，不断增加业务，财政压力加剧。面对这些情况，去年9月，58同城强制要求员工实习“966工作制（即早上九点上班，晚上九点下班，一周上六天班）”，大量员工因此跑到姚劲波的微博下喊话。期间辞职的人也不在少数，甚至有段子说：58的业务员在给别人推销的时候，被问听说你们要加班啊，要不要来我们这里，福利特别好，于是58业务员就跳槽了。

今年，58则直接采取裁员制度。发内部邮件称要将淘汰率从原来的5%-10%，提高到现在的10%，并且被主动淘汰的人的薪资的一半，也会放在部门的加薪预算里。而如果你在公司的平均工作时长不在公司同类部门工作时长排前一半的话，公司会不考虑HC预算，由人事部门直接执行。

准备离职的员工表示：想裁员直接说，不用强行找理由。

早年间，姚劲波在接受采访时，曾提及58同城不会站队BAT，始终坚持自己独立发展。没想到一个月以后，一个巨大的巴掌打向了自己。腾讯以7.36亿美元，拿下了58同城19.9%的股份，成为了58同城第一大投资机构。话音还没落的姚劲波就这样一头扎进了腾讯的怀抱。

尽管姚劲波一直强调，接受投资并不等于站队腾讯，到现在为止，可能也就他自己一个人还相信这句话吧。

继优酷土豆、快的滴滴之后，又一对不可能在一起的公司合并了。也就是58同城和赶集网。还记得当优酷和土豆合并时，姚劲波曾说“我不可能合并一家没有任何意义的公司”，如今，这家公司的总裁正和他一起工作。

58的员工评价姚劲波，是一个十分重视外在形象的人，每天到公司的第一件事不是打开邮件处理公务而是打开百度搜索“姚劲波”。就是这样一个注重形象的人，竟然会说出这样的一番话，足以证明当时赶集网和58同城的竞争有多激烈。

但是再激烈的竞争，最后的目的还是为了利益。背负着华尔街与腾讯压力的姚劲波，如果和赶集网合并，能够获取更多的利益，自然是愿意拉下脸面的，反正打脸也不是第一次了。
